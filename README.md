# 🌍 Ishwar Holidays – Official Website

Welcome to the official repository for **Ishwar Holidays**, a travel and tourism platform designed to provide seamless booking, enquiries, and travel experiences.

🔗 Live Website: https://ishwarholidays.com/

---

## 📌 About the Project

**Ishwar Holidays** is a modern travel website that enables users to:

* Explore travel packages and destinations
* Submit booking requests
* Send enquiries directly to the admin team
* Receive automated confirmations via email
* Experience a responsive and user-friendly interface

The platform is built with performance, simplicity, and user experience in mind.

---

## 🚀 Features

* ✈️ Travel package browsing
* 📩 Booking & enquiry forms
* 📧 Automated email notifications (admin + customer)
* 🔥 Firebase integration for backend services
* 📱 Fully responsive design
* ⚡ Fast and optimized frontend

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend Services:** Firebase
* **Email Service:** EmailJS
* **Hosting:** Firebase Hosting / Web Hosting

---

## 🔐 Security & Configuration

This project uses:

* Firebase for app configuration and backend services
* EmailJS for sending transactional emails

⚠️ Note:
Public configuration keys (e.g., Firebase config) are safe for client-side use, but backend rules and access controls must be properly secured.
